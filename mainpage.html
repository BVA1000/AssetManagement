ASSET MANAGEMENT APPLICATION

This is the Asset Management Application. It was designed for organizing the total value of your assets. What you have to do is type in the value of each asset (in USD) in each category. Once you type out the value of each category, you press the submit button and it will create a list of each asset's value and will also calculate the grand total of all your asset values combined.
